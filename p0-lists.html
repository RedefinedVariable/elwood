RedefinedVariable's Project 0: Nested List

Nested Lists via nested scss

  1. Convert scss file to css file
    1. Download SASS from GitHub
      • Unzip
      • Add to Path to tell windows where dart-sass is at.
      • Open cmd window and check the version using sass --version
    2. Create a .scss file
    3. Compile the .scss file to the .css file by running the sass command
      • sass variable.scss ..\css\styles.css
    4. Check your results
  2. After editing the .scss file recompile it to update the .css file
  3. If it is not compiling, check the paths to the files used in the command.
